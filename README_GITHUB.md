# Cosmic Collector

🚀 **Live Demo:** [https://cosmic-collector.vercel.app](https://cosmic-collector.vercel.app)

*An addictive arcade space game with NFT integration powered by Hedera Hashgraph*

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/mikk69/cosmic-collector)

## 🎮 Game Features

- **Addictive Gameplay** - Classic arcade-style space shooter
- **NFT Integration** - Collect unique spaceship NFTs
- **Hedera Blockchain** - Powered by Hedera Hashgraph testnet
- **Realistic Physics** - Advanced physics engine with particle effects
- **Progressive Difficulty** - Dynamic scaling for endless fun
- **Leaderboards** - Blockchain-based global rankings

## 🚀 Quick Start

### Play Online
Visit: **[cosmic-collector.vercel.app](https://cosmic-collector.vercel.app)**

### Run Locally
```bash
# Clone the repository
git clone https://github.com/mikk69/cosmic-collector.git
cd cosmic-collector

# Start local server
python -m http.server 8000
# OR
npx http-server

# Open browser
open http://localhost:8000
```

## 🎯 How to Play

### Controls
- **Move:** Arrow Keys or WASD
- **Shoot:** Spacebar
- **Pause:** ESC

### Objective
1. Survive waves of enemies
2. Collect cosmic gems for points
3. Upgrade your weapons
4. Buy NFT spaceships
5. Compete on global leaderboards
